Difference between var, let, const ?

The scope of a var variable is functional scope.So it can be updated and re-declared into the scope.
On the other hand, The let variable is block scope.It can not be updated or re-declared outside the scope but can be update / modify inside the scope.
Finally The const variable is also block scope like let but It can not be updated or re-declared into the scope.

Difference between arrow and regular function ?

1. Using 'new' keyword
Regular functions created using function declarations or expressions are constructible and callable.So they can be called using the "new" keyword.
However, the Arrow functions are only callable and not constructible. Hence, they can never be invoked with the "new" keyword.

2. Use of 'this' keyword
The behavior of 'this' inside of an arrow function differs considerably from the regular function's 'this' behavior. The arrow function doesn't define its own execution context.

Why use template string ?

Template string also called Backticks (``) are an ES6 feature that allows us to create strings in JavaScript.

  • 2. Can write multiline without using the new line character
